# TrailLens 文档

> 把一整次徒步的素材丢进去 → AI 自动选片、点评、生成游记,并规划你下次的拍摄计划。

- **首次接触**:先读 [快速开始](/getting-started)。
- **想看架构**:[架构](/architecture) → [Agents](/agents)。

- **想贡献代码**:[贡献](/contributing)。
