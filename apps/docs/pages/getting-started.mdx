# 快速开始

> 5 分钟跑通骨架(无 GPU、无 API key、无装包)。

```bash
git clone https://github.com/lzf00/traillens
cd traillens
make verify       # 78 个测试 + agent demo + 美学评估自检
```

如果你想看流式 SSE:

```bash
# Terminal A
make api

# Terminal B
curl -N -X POST http://localhost:8000/v1/trails \
  -H 'Content-Type: application/json' \
  -d '{"name":"demo"}' | jq -r .id | \
xargs -I {} curl -N -X POST http://localhost:8000/v1/trails/{}/run
```

看到一连串 `event: ...` 推过来就成功了。
