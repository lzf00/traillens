今天做了个有意思的小实验:让豆包给我 5 张风光照打 8 维分数,然后我自己再校准。

数据(豆包 → 我):

```
#1  8.3 → 9.0  ↑ +0.7   ← 豆包低估了我的最爱
#2  7.8 → 7.5  ↓ -0.3
#3  7.8 → 8.0  ↑ +0.2
#4  7.1 → 6.5  ↓ -0.6   ← 豆包高估了 我更挑剔
#5  7.5 = 7.5         ← 完全一致
```

平均带符号偏差 +0.00(无系统偏向),平均绝对偏差 ±0.36(对齐很好)。

**3 个观察:**

1️⃣ 豆包的风光评估**够用** — 整体方向没错。
2️⃣ 但有个性化空间 — #1 和 #4 显示我看到了豆包看不到的细节。
3️⃣ 5 张数据当然不够训,但**够验证整套流水线**。
