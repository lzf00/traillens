## 正文(800 字,小红书算法友好)

最近在做一个个人 AI 项目,今晚出了第一个**真实可信的数据点**,
记录一下 🍃

---

🎯 **我做了什么**

让豆包(火山方舟)的视觉大模型,给我自己的 5 张风光照,
按 8 个维度打分(构图 / 视觉 / 技术 / 原创 / 主题 / 情感 / 格式塔 / 综合)。

然后我**不看 AI 答案**,凭直觉自己再打一次。

---

📊 **结果意外地有意思**

```
       豆包         我          差
  #1   8.3   →   9.0       ↑ +0.7
  #2   7.8   ←   7.5       ↓ -0.3
  #3   7.8   →   8.0       ↑ +0.2
  #4   7.1   ←   6.5       ↓ -0.6
  #5   7.5   =   7.5       (一致)
```

**关键数字**:

- 🎯 **平均带符号偏差: +0.00** ← AI 没有系统性高估/低估
- 🎯 **平均绝对偏差: ±0.36** ← 比想象的小很多
- 🎯 **3/5 的差异 < 0.5 分** ← 算"对齐"门槛内

---

✨ **3 个发现**

**1. 豆包对风光的整体评估「够用」**

平均偏差只有 0.36(满分 10),远小于 1.0 这个"显著差异"门槛。
意味着不需要做出"另一个 GPT-5V 替代品"——豆包已经能干这事 80%。

**2. 但有 2 张照片显示出"我看到的 AI 看不到"**

- #1 我给 9.0 豆包给 8.3 → AI 没看出这张为什么是我的"心头好"
- #4 我给 6.5 豆包给 7.1 → 我比 AI 更挑剔,可能看到了它没看到的瑕疵

这就是为什么 PIAA(个人美学)是真问题——每个人都有自己的偏好。

**3. 用户体验设计比想象的难**

第一次让自己测标注 UI,结果**没看大图就保存了所有照片**🤣
分数全是豆包的默认值——这个"懒"必须假设最大化。
